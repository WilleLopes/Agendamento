
## Tech Stack


**Server:** Node, Express, JWT, Middlewares, Prisma, Postgresql


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file


`DATABASE_URL=""`

`ACCESS_TOKEN_SECRET=""`

`ACCESS_KEY_TOKEN = ""`

`ACCESS_KEY_TOKEN_REFRESH = ""`
